## Problem Overview
 
Organisations relying on manual or fragmented goal-tracking such as spreadsheets, emails, and offline review cycles face three core problems:
 
- **No visibility:** Managers cannot monitor team progress in real time
- **No alignment:** Employees lack clarity on how their work connects to organisational priorities
- **No accountability:** HR teams piece together data at appraisal time with no audit trail

## Project Goals
To design and develop a full-stack web portal from scratch (within a hackathon timeline), covering three distinct user roles, validation logic, quarterly tracking windows, and reporting.
GoalSync is a structured, digital portal that supports the full lifecycle of employee goals:
 
```
Goal Creation → Manager Approval → Goal Locking → Quarterly Check-ins → Analytics & Export
```
